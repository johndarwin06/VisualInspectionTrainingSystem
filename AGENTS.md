# Visual Inspection Training System

Framework:
- C#
- WPF
- .NET Framework 4.8
- MVVM

Rules

- Modify files directly.
- Rewrite entire files.
- Never create duplicate classes.
- Preserve architecture.
- Compile after every modification.
- Fix compile errors before continuing.
- Use #region.
- Use XML documentation.

Workflow

1. Analyze
2. Edit
3. Save
4. Build
5. Report
6. Wait for next task
